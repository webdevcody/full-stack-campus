# Community Posts Epic

## Overview
This epic contains all features related to community posting functionality. Users can create posts, comment on posts, react to content, and upload media attachments.

## Features (19 total)
- **01-06**: Post creation, viewing, editing, and deletion
- **07-11**: Comment system with nested replies
- **12-14**: Reaction/like system for posts and comments
- **15-17**: File attachments (images and videos)
- **18**: Admin post pinning
- **19**: Category filtering

## Implementation Order
1. Start with feature 01 (database schema)
2. Build post CRUD features (02-06)
3. Add comment system (07-11)
4. Add reactions (12-14)
5. Add attachments (15-17)
6. Add admin and filtering features (18-19)

## Dependencies
- None (foundation epic)
