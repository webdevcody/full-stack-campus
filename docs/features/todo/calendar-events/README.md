# Calendar Events Epic

## Overview
This epic contains all features related to community calendar and event management. Admins can create events and members can view them on a calendar.

## Features (3 total)
- **26**: Event table schema
- **27**: Admin can create events
- **28**: User can view calendar events

## Implementation Order
1. Start with feature 26 (event table)
2. Build event creation (27)
3. Build calendar view (28)

## Dependencies
- Independent epic, can start anytime
- Requires Admin epic (feature 47) for admin permissions
