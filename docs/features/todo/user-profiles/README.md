# User Profiles Epic

## Overview
This epic contains all features related to user profile management. Users can create rich profiles with bios, skills, and portfolio items.

## Features (6 total)
- **20**: User profile table schema
- **21**: Edit profile bio
- **22**: Add skills to profile
- **23**: Portfolio item table schema
- **24**: Add portfolio items
- **25**: View public profiles

## Implementation Order
1. Start with feature 20 (user_profile table)
2. Build profile editing (21-22)
3. Add portfolio system (23-24)
4. Add public profile viewing (25)

## Dependencies
- Can start independently, but integrates with Community Posts and Messaging epics
